# emptycup_Assignment
A dashboard with list of designer with there details like price, past projects, rantings etc.

# How to start in local environment
here backend and frontend are in one folder but run them saparatly. 
- For Frontend :
    1. npm install
    2. npm run dev
- Backend :
    1. npm intall
    2. node index.js

 # Tech useed : 
   - Frontend :
       - React.js
       - bootstrap
       - Material UI
    - Backend :
       - NodeJs
       - MongoDB
       - ExpressJs
